# sensor2IoTHub
## The Repository
こちらのリポジトリでは、テレメトリデータをAzure IoT Hubに送るシミュレータを提供する。送信するテレメトリデータを次に示す。
- timeseries(時刻)  
  - ex.) 2022/06/06 23:47:01
- temperature(温度) 
  - ex.) 69.35
- humidity(湿度)
  - ex.) 97.91
